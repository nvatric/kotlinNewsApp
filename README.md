#  Kotlin News App

Android application developed in Kotlin for browsing, filtering and saving news articles.

---

## About The Project

Kotlin News App is a mobile application built as an academic project focused on Android development and modern mobile application architecture.

The application allows users to browse news articles by categories, search content, save favorite articles locally and interact with a responsive user interface.

The project includes API integration, local database storage and modern Android development practices using Kotlin.

---

##  Features

* Browse news articles from different categories
* Search and filter news content
* Save favorite articles locally
* Modern and responsive Android UI
* API integration for fetching news data
* Local database support using Room
* Image loading and network communication

##  Project Structure

The project is organized into:

* `data/` → database, API and repository-related classes
* `network/` → API communication and services
* `ui/` → activities, fragments and user interface components
* `database/` → Room database configuration and DAO classes

##  How To Run

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle dependencies
4. Run the application on an emulator or Android device

```bash
git clone https://github.com/nvatric/kotlinNewsApp.git
```
